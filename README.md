# LockerManagement

How to install:

1. Restore Nugget Package
2. Create new database with name "LockerManagement" in "localhost\SQLEXPRESS"
3. Run this query berikut
    - DAL\GMDS.edmx.sql
    - DAL\LockerContext.edmx.sql
    - DAL\Seed.sql
4. Run the project

Technologies:

Language: C#
Framework: 
  - ASP.Net MVC
  - Entity Framework Database First
